API Endpoints

GET /v2/current?lat={lat}&lon={lon} — Current weather conditions
GET /v2/forecast?lat={lat}&lon={lon}&hours={hours} — Hourly forecast up to 72h
GET /v2/grid?bbox={minLon},{minLat},{maxLon},{maxLat} — Bulk GeoJSON data for area
GET /v2/alerts?lat={lat}&lon={lon}&radius_km={radius} — Weather alerts
